4 2. Genetic Algorithm Steps
4- Generate random population
This function generates a random population of a given size
[4]: # Generate a random individual (solution)
def create_individual():
return [random.randint(0, 1) for _ in range(len(items))]
• Each individual (solution) is a binary list (0 or 1):
– 1 = Take the item
– 0 = Leave the item
• Example: [1, 0, 1, 0, 0, 1] means take items 1, 3, and 6.
5- Calculate chromosome fitness
This function is used to calculate the fitness of a chromosome. It takes a chromosome as an
argument and iterates through it.
[5]: # Calculate fitness (total value, penalize if overweight)
def fitness(individual):
total_weight = sum(items[i][0] * individual[i] for i in range(len(items)))
total_value = sum(items[i][1] * individual[i] for i in range(len(items)))
if total_weight > max_weight:
return 0 # Penalize invalid solutions
return total_value
• Computes total value of selected items.
• If total weight > max_weight, fitness = 0 (invalid solution).
• Example:
– [1, 1, 0, 0, 0, 0] → Weight = 1+2=3, Value = 2+4=6 → Fitness = 6 ￿
– [1, 1, 1, 1, 1, 1] → Weight = 21 (too heavy) → Fitness = 0 ￿

[6]: # Select parents using tournament selection
def select_parent(population):
tournament = random.sample(population, 3) # Pick 3 random individuals
return max(tournament, key=fitness) # Best one becomes parent
7- Perform crossover
This function performs crossover between two chromosomes.
[7]: # Single-point crossover
def crossover(parent1, parent2):
point = random.randint(1, len(items) - 1)
child = parent1[:point] + parent2[point:]
return child
• Single-point crossover:
– Split two parents at a random point and combine.
– Example:
∗ Parent 1: [1, 0, 1 | 0, 0, 1]
∗ Parent 2: [0, 1, 0 | 1, 0, 0]
∗ Child: [1, 0, 1, 1, 0, 0]

[8]: # Mutation (flip a random bit)
def mutate(individual):
for i in range(len(individual)):
if random.random() < mutation_probability:
individual[i] = 1 - individual[i] # Flips 0→1 or 1→0
return individual
• Each bit flips (0→1 or 1→0) with a 20% chance.
• Example:
– Before: [1, 0, 1, 0, 0, 1]
– After: [1, 1, 1, 0, 0, 1] (2nd bit flipped)
